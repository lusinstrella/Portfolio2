Quick Reference: Picture Element Pattern


<picture>
    <source srcset="path/to/image.webp" type="image/webp">
    <img src="path/to/image.jpg" alt="description" class="existing-class" loading="lazy">
</picture>

Remember:

  • Keep all existing classes, IDs, and attributes on the <img> tag
  • The <source> only needs the WebP path and type
  • Modern browsers use WebP, old browsers use JPG/PNG fallback
  • Don't delete original JPG/PNG files - they're the fallback!

Testing Checklist

  1. ✅ Convert all images to WebP
  2. ✅ Update HTML with picture elements
  3. ✅ Test in Chrome (should load WebP)
  4. ✅ Test in Safari (should load WebP)
  5. ✅ Verify all images display correctly
  6. ✅ Check DevTools Network tab for WebP loading
  7. ✅ Run Lighthouse performance test
  8. ✅ Compare before/after page sizes
